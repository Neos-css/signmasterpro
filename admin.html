📄 ผลงานที่ AI ร่างให้

🎬 ทำคลิปผลงาน (โพสต์ TikTok / Reels)

เลือกหมวด → ใส่ข้อความหัวคลิป → AI สร้างคลิปแนวตั้งจากผลงานจริงให้ (ฟรี · ไม่เปลือง API · อักษรไทยคมชัด)
🖼️ เลือกรูปที่อยากใส่ (กดได้หลายรูป · ไม่เลือก = สุ่มให้ 5 รูป)
⏱️ สร้างคลิปใช้เวลา ~15 วินาที (อัดเรียลไทม์) · ได้ไฟล์ .webm · เปิดหน้านี้ค้างไว้จนเสร็จ

⬇️ ดาวน์โหลดคลิป

🎥 AI วิดีโอ (เชื่อมบริการภายนอก)

สร้างวิดีโอ AI จริงจาก prompt + รูป (คุณภาพสูง · มีค่าใช้จ่าย) — ฟอร์มพร้อมแล้ว เหลือแค่เลือกผู้ให้บริการ + ใส่ API key ใน worker
⚠️ ยังไม่ได้เชื่อม API — เลือกเจ้า + ใส่ key ใน worker ก่อนถึงสร้างได้จริง (ตอนนี้กดได้ แต่จะแจ้งว่ายังไม่พร้อม)
