# Signmaster Pro — เว็บตัวอย่าง (Demo)

ศูนย์ป้ายโฆษณาครบวงจร — บูธ · อิงค์เจท · ป้ายทุกชนิด · ตู้ไฟ
พร้อมระบบสั่งทำงาน + ประเมินราคาด้วย AI ("น้องอิงค์")

> เว็บตัวอย่างนี้เป็น static site (HTML/CSS/JS) · พัฒนาโดยทีม **Neo-S AI**

## โครงสร้างไฟล์
- `index.html` — หน้าเว็บหลัก (Hero · บริการ · ผลงาน · สั่งทำงาน · เกี่ยวกับเรา · ติดต่อ)
- `price-config.js` — ตารางราคา (แก้ราคาที่ไฟล์นี้ที่เดียว)
- `gallery-data.js` — รายการรูปผลงานในแกลเลอรี
- `media/` — รูปภาพทั้งหมด

## เปิดดู
ดับเบิลคลิก `index.html` หรือเปิดผ่าน GitHub Pages
